# 原型截图说明

本文件夹用于存放 AI 剧场客服管家的脱敏原型截图。截图仅用于展示产品路径和交互逻辑，不包含真实公司、真实项目、真实角色、真实剧情文本、票务规则或内部业务数据。

## 截图清单

| 文件 | 页面含义 | 展示重点 |
| --- | --- | --- |
| `splash.png` | 进入页 | 建立项目氛围，展示用户进入 AI 服务前的第一触点 |
| `feature-entry.png` | 服务入口页 | 展示售前咨询、角色推荐、体验后对话三个主要入口 |
| `role-quiz.png` | 角色初测页 | 展示体验前如何通过轻量测评收集用户偏好 |
| `recommendation-result.png` | 推荐结果页 | 展示系统如何给出推荐角色、匹配理由和下一步行动 |
| `post-experience-chat-list.png` | 体验后对话列表 | 展示体验后用户可继续选择角色进行复盘或互动 |
| `post-experience-chat-demo.png` | 对话示例页 | 展示 AI 在体验后对话中承接线索追问和情绪陪伴 |

## 脱敏原则

- 项目名、角色名、地点名均已替换为虚构表达。
- 截图只保留产品结构和交互关系，不展示真实业务素材。
- 涉及真实剧情、真实运营规则、真实票务信息的内容不在公开仓库中展示。
- 后续如果新增截图，应先检查是否包含可识别真实项目的信息，再放入本文件夹。

## 推荐引用顺序

在项目说明或展示文章中，建议按下面顺序引用：

1. `splash.png`
2. `feature-entry.png`
3. `role-quiz.png`
4. `recommendation-result.png`
5. `post-experience-chat-list.png`
6. `post-experience-chat-demo.png`

这个顺序对应用户从进入服务、选择能力、完成推荐，到体验后继续对话的完整路径。
