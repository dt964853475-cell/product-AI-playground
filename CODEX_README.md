# product-ai-playground

这个仓库用于记录 AI 产品方向的个人实践，包括产品原型、AI 工作流、智能体设计、评测方法和项目复盘。

当前内容以 3 个项目为主：

- 线下体验服务 AI 管家：将售前咨询、体验前推荐、体验后复盘等服务流程整理成 AI 管家产品方案。
- 拍即合：面向法拍车新手用户的出价辅助小程序 MVP。
- 对话模拟智能体：面向新人学习、角色理解和话术练习的知识问答智能体。

## 内容原则

- 只展示脱敏后的产品思路、流程和方法。
- 不包含前司名称、真实项目名称、真实角色名称、内部业务数据或未公开资料。
- 每个项目尽量说明：业务问题、目标用户、产品方案、核心流程、AI 使用方式、验证结果和下一步计划。
